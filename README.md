# robosys2022a

これらのプログラムは授業により製作したものです。



## 内容について

* **LICENSE**
(著作権についての文章です。)

* **.github/workflow**
(テストバッジのプログラムです。)

* **test.bash**
(`register.py`についてのテストプログラムです。)

* **register.py**
(入力した値を足していき、最終的に合計値と消費税10％とした場合の合計値内の税金の値を出すプログラムです。　　
基本的には`./register.py`などの後に入れたい数字を順に入力する様になっています。
また、入力には`sys.argv`を使っているので、それに対応する入力のやり方で>あれば大丈夫です。)

->入力例
```
$ ./register.py 10 20 30
合計金額は60円（内税金6.0円）
```

## このプログラムの使用方法について

１．ubuntu22.04.1 LTSを開いて任意の名前のディレクトリを作成します。ディ>レクトリを作成しなかった場合は現在開かれている場所にクローンしてきたリポジトリが作成されます。
__(対応しているPythonのバージョンを一度確認してください。)__

２．リポジトリの各種プログラムをみることができる画面の右くらいにある緑色の`<>Code`というボタンをクリックします。

３．`clone`という文字の下に`HTTPS` `SSH` `GitHub CLI`と書いてある部分の`SSH`を選択して下に出てきたリポジトリパスをコピーします。

４．（１.）で作成したディレクトリで
```
$ git clone git@github.com:moritaddaiki/robosys2022a.git
```
を実行します。

５．これでリポジトリのクローンが完了しました。後は各コードを実行すれば動きます。
